# Durable Charts

Collaborative charts editor with durable objects.

## Components and deploy

- Fronend is CF pages
- Backend is CF durable objects

### Pages

Merge your changes into master and CF pages will do the rest.
	
	git push

### Workers and durable objects

	cd workers/
	wrangler publish
